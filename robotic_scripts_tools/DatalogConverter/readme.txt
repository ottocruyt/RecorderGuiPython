Rack Conversion tools
for further questions contact
Matthias Haase <matthias.haase@still.de>

This python scripts contain functionalities to convert different files from or to the rack datalog format to another format.
Available scripts:

-CameraToVideo.py
convert recorded camera data to a series of images or a video file

-PcdToRack.py
select a folder with pcd files and convert them to scan3d data that can be replayed with the rack data player.

-Recog3dToVideo.py
convert a set of recordings (scan3d and obj_recog) to a video with based on the scan3d intensity or distance with object overlays
